# borechat
TUI chat app originally using the bore.pub service, now using the better protocol Matrix.

## Installation
Run `pip install requests`, install client.py and run it with `python3`. (3.10+ sufficient)

If on Windows or MacOS install python from [the downloads page](https://www.python.org/downloads/)

If on Linux install the python3 package using your package manager

## Getting started
Once inside Borechat you can enter `/help`.
